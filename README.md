# omg-web

Website, documentation site, and web workers for OMG.

## Structure

- `site/` - marketing site and web app surface
- `workers/router/` - routes docs traffic to Docusaurus (omg-docs.pages.dev)
- `workers/` - standalone Cloudflare workers
